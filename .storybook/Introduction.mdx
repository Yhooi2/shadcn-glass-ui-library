## ğŸš€ Quick Start

### Installation

```bash
# Via shadcn CLI (recommended)
npx shadcn@latest add @shadcn-glass-ui/button

# Via npm
npm install shadcn-glass-ui
```

### Basic Usage

```tsx
import { ButtonGlass, InputGlass, GlassCard } from 'shadcn-glass-ui';

export default function App() {
  return (
    <div className="theme-glass p-8">
      <GlassCard variant="glass" intensity="medium">
        <h2>Welcome</h2>
        <InputGlass placeholder="Enter text..." />
        <ButtonGlass variant="primary">Submit</ButtonGlass>
      </GlassCard>
    </div>
  );
}
```
