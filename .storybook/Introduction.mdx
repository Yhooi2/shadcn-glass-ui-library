## What's New in v2.0.0

### üé® 3-Layer Token Architecture

Create custom themes **90% faster** (10-15 minutes vs 2-3 hours):

- **Layer 1**: 207 primitive OKLCH tokens (`--oklch-purple-500`, `--oklch-white-95`)
- **Layer 2**: Semantic tokens (`--semantic-primary`, `--semantic-surface`)
- **Layer 3**: Component tokens (`--btn-primary-bg`, `--modal-bg`)

**Zero hardcoded OKLCH values** in all themes. See [TOKEN_ARCHITECTURE.md](https://github.com/Yhooi2/shadcn-glass-ui-library/blob/main/docs/TOKEN_ARCHITECTURE.md) for details.

### üÜï New Components

- **StepperGlass** - Compound stepper for multi-step workflows (numbered, icon, dots variants)

### ü™ù Exported Hooks

Now available for your custom components:

```tsx
import { useFocus, useHover, useResponsive, useWallpaperTint } from 'shadcn-glass-ui';
```

### ‚ö†Ô∏è Breaking Changes

**CSS Variables Renamed** (semantic naming):

- `--metric-emerald-*` ‚Üí `--metric-success-*`
- `--metric-amber-*` ‚Üí `--metric-warning-*`
- `--metric-blue-*` ‚Üí `--metric-default-*`
- `--metric-red-*` ‚Üí `--metric-destructive-*`

See [Migration Guide](https://github.com/Yhooi2/shadcn-glass-ui-library/blob/main/docs/migration/CSS_VARIABLES_MIGRATION_2.0.md).

```tsx
  return (

    </GlassCard>
  );
}
```
