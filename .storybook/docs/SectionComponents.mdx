## Usage Examples

### HeaderNavGlass

```tsx
import { HeaderNavGlass } from 'shadcn-glass-ui';

<HeaderNavGlass
  logo={<Logo />}
  navItems={[
    { label: 'Dashboard', href: '/dashboard' },
    { label: 'Projects', href: '/projects' },
    { label: 'Settings', href: '/settings' },
  ]}
  user={{
    name: 'John Doe',
    avatar: '/avatar.jpg',
  }}
  onSearch={(query) => console.log(query)}
/>
```

### ProfileHeaderGlass

```tsx
import { ProfileHeaderGlass } from 'shadcn-glass-ui';

<ProfileHeaderGlass
  user={{
    name: 'Jane Developer',
    username: '@janedev',
    avatar: '/jane.jpg',
    bio: 'Full-stack developer passionate about React',
    location: 'San Francisco, CA',
    stats: {
      followers: 1234,
      following: 567,
      repos: 89,
    },
    languages: [
      { name: 'TypeScript', level: 'native' },
      { name: 'Python', level: 'fluent' },
    ],
  }}
/>
```

### RepoCardGlass

```tsx
import { RepoCardGlass } from 'shadcn-glass-ui';

<RepoCardGlass
  repo={{
    name: 'awesome-project',
    description: 'A really awesome project',
    stars: 1234,
    forks: 56,
    language: 'TypeScript',
    updatedAt: '2024-01-15',
  }}
  expanded={false}
  onToggle={() => setExpanded(!expanded)}
/>
```

### TrustScoreCardGlass

```tsx
import { TrustScoreCardGlass } from 'shadcn-glass-ui';

<TrustScoreCardGlass
  score={87}
  metrics={[
    { label: 'Code Quality', value: 92, trend: 'up' },
    { label: 'Activity', value: 78, trend: 'stable' },
    { label: 'Community', value: 85, trend: 'up' },
  ]}
/>
```

## Building Pages

Section components combine easily to create complete pages:

```tsx
function ProfilePage() {
  return (
    <div className="min-h-screen">
      <HeaderNavGlass {...headerProps} />

      <main className="container mx-auto py-8">
        <ProfileHeaderGlass {...profileProps} />

        <div className="grid grid-cols-1 md:grid-cols-2 gap-6 mt-8">
          <TrustScoreCardGlass {...scoreProps} />
          <CareerStatsGlass {...careerProps} />
        </div>

        <FlagsSectionGlass {...flagsProps} />

        <h2 className="text-xl font-semibold mt-8 mb-4">
          Repositories
        </h2>
        {repos.map(repo => (
          <RepoCardGlass key={repo.name} repo={repo} />
        ))}
      </main>
    </div>
  );
}
```
