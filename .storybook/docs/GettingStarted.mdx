# Getting Started

Quick start guide for Glass UI.

## Installation

### Via shadcn CLI (recommended)

<Source code={`npx shadcn@latest add https://shadcn-glass-ui.dev/r/button-glass`} language="bash" />

### Via npm

<Source code={`npm install shadcn-glass-ui`} language="bash" />

### Via yarn

<Source code={`yarn add shadcn-glass-ui`} language="bash" />

## ThemeProvider Setup

Wrap your application with ThemeProvider:

<Source
  code={`import { ThemeProvider } from 'shadcn-glass-ui';

function App() {
  return (
    <ThemeProvider defaultTheme="glass">
      <YourApp />
    </ThemeProvider>
  );
}`}
  language="tsx"
/>

## CSS Import

Add the required styles to your main application file:

<Source
  code={`// In main.tsx or App.tsx
import 'shadcn-glass-ui/dist/styles.css';`}
  language="tsx"
/>

## Basic Usage

<Source
  code={`import { ButtonGlass, InputGlass, GlassCard } from 'shadcn-glass-ui';

function MyComponent() {
  return (
    <GlassCard variant="glass" intensity="medium">
      <h2>Hello, Glass UI!</h2>
      <InputGlass
        placeholder="Enter text..."
        className="mb-4"
      />
      <ButtonGlass variant="primary">
        Submit
      </ButtonGlass>
    </GlassCard>
  );
}`}
  language="tsx"
/>

## Theme Switching

Use the `useTheme` hook for programmatic theme switching:

<Source
  code={`import { useTheme } from 'shadcn-glass-ui';

function ThemeSwitcher() {
  const { theme, setTheme, cycleTheme } = useTheme();

  return (
    <div>
      <p>Current theme: {theme}</p>
      <ButtonGlass onClick={() => setTheme('aurora')}>
        Aurora
      </ButtonGlass>
      <ButtonGlass onClick={cycleTheme}>
        Next Theme
      </ButtonGlass>
    </div>
  );
}`}
  language="tsx"
/>

## shadcn/ui Compatibility

Glass UI is fully compatible with shadcn/ui API:
